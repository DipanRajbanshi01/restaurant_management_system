# Logo Image

Please place your restaurant logo image in this directory as `logo.png`.

The logo should be:
- Named: `logo.png`
- Format: PNG (recommended for transparency)
- Recommended dimensions: 200x200px or similar aspect ratio
- The logo will be displayed in the navbar across all pages

The logo will automatically navigate users to their respective dashboards based on their role:
- Admin → Admin Dashboard
- Chef → Chef Dashboard
- User → User Dashboard
- Not logged in → Home page
